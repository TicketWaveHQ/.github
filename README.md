# TicketWaveHQ / .github

This repository hosts the **TicketWave HQ Ltd** organisation profile and org-wide community health files for [github.com/TicketWaveHQ](https://github.com/TicketWaveHQ).

The org-profile page itself is rendered from [`profile/README.md`](profile/README.md) — that is the one to read if you want a tour of what the company does.

## What's in here

| File | Purpose |
|---|---|
| [`profile/README.md`](profile/README.md) | Public org-profile page rendered at github.com/TicketWaveHQ |
| [`SECURITY.md`](SECURITY.md) | Responsible-disclosure policy (RFC 9116) |
| [`CODE_OF_CONDUCT.md`](CODE_OF_CONDUCT.md) | Contributor Covenant 2.1 |
| [`CONTRIBUTING.md`](CONTRIBUTING.md) | How to contribute to public repos in this org |
| [`SUPPORT.md`](SUPPORT.md) | Where to get help |
| [`.github/FUNDING.yml`](.github/FUNDING.yml) | Funding configuration (currently empty) |
| [`.github/ISSUE_TEMPLATE/`](.github/ISSUE_TEMPLATE/) | Default issue templates inherited by repos without their own |

## Why org-wide files live here

GitHub serves community health files in a repo named `.github` (or `docs/` of any repo) as **defaults** for every repository in the organisation. Individual repos can override any file by adding their own.
